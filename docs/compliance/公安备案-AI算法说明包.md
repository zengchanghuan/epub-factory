## 二、算法来源说明（核心）

本公司 **不自行研发、不训练、不微调、不部署** 任何大语言模型或深度合成算法。

FixEpub 的 **AI 翻译** 功能（电子书文本翻译、双语对照排版等）仅通过 **HTTPS API** 调用第三方已备案服务提供方的模型能力；推理计算在对方服务器完成，本公司不持有模型权重、不进行二次训练。

**调用方（本公司）**：深圳市长欢文化艺术有限公司  
**技术服务方（模型提供方）**：北京深度求索人工智能基础技术研究有限公司（品牌：DeepSeek）  
**接入方式**：DeepSeek 开放平台（platform.deepseek.com）API，注册主体认证后获取 API Key 调用

## 五、本公司合规承诺

1. **不自研算法**：不训练、不微调、不对外提供自研大模型。  
2. **场景限定**：AI 功能仅用于用户上传电子书的翻译处理，不生成违法不良信息；格式转换功能不依赖生成式大模型。  
3. **用户告知**：网站「关于我们」及用户协议中已说明 AI 功能由 DeepSeek API 提供、内容由模型生成需用户甄别。  
4. **数据与隐私**：上传文件仅用于当次处理，处理完成后删除；详见网站《隐私政策》。  
5. **后续义务**：如属地网信部门要求办理「调用已备案大模型」的 **大模型登记**，本公司将按规定另行向省级网信办申报（与本次公安网站备案为不同事项）。
